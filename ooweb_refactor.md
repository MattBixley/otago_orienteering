# website refactor

# tasks
your task is to refactor the Dunedin Orienteering Club website into a more user firendly and appealing space.

Dunedin Orienteering has renamed to Otago Orienteering and want to rebuild their website to be more user friendly.
1) research and review the current website https://www.dunedinorienteering.org/
2) compare the current wesbite to other New Zealand Clubs. PAPO and Auckland are given in resources
3) where training is discussed don't build new pages but reference to relevent Orienteering New Zealand resources
4) build a new github repo and github pages website as a draft of what it could be. setup a good directory structure and pages layout
5) review what website publishing resources are available and select 3 that provide simple admin interfaces to enable several people to add and change information.

# contents
the new website should contain the following
about - this will be updated with what the club is and the clubs goals and aspirations
events - what events are upcoming, there should be a calendar or calendat list. each event should then be able to be cliked to a new link with more information.
policy - the clubs policies
resources - training materials
results - closley linked to the events pages. results should be categorised by year to begin with and then events type.
